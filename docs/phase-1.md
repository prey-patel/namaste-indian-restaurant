# Phase 1: Project Setup & Architecture Foundation

This phase establishes the baseline codebase for Namaste Indian Restaurant.

## What is Included
*   **Next.js App Router Scaffolding:** Configured with strict TypeScript and Tailwind CSS v3.
*   **shadcn/ui setup:** Installed `button` and `card` components.
*   **next-intl localization:** Configured for `pl` (default) and `en`.
*   **Supabase client helpers:** Segmented into browser-client, server-client (using cookies), and admin-client.
*   **Routing paths config:** Whitelisted routes mapping in `lib/routes/path.ts`.
*   **Public Page Templates:** Home, Menu, Reservations, Our Story, Contact, My Status, Privacy Policy, Cookie Policy, and Terms of Service.
*   **Admin Panel Layout Templates:** Main dashboard mique and login page placeholders.
*   **Architecture skeletons:** Code skeletons with interfaces for pricing calculations, geocoding fallbacks, and notification retries.

## What is NOT Included (Deferred to Later Phases)
*   **Phase 2–10 features:** No actual database tables, no manual/automatic reservations, no cart/orders, and no real geocoding or email APIs are integrated.
